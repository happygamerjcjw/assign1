Animations

We added animations into some of the objects within the webpages. For example, the company logo on the top left which eases in and out when you hover over it with the mouse. There are also some intro animations for the main titles.


  • Effect: When a user hovers over the logo, it slightly enlarges (zooms in) to 110% of its original size.
  • Purpose: This effect draws attention to the logo, encouraging user interaction and signaling that the element is clickable.

Improved form layout

We utilised a more cleaner and professional design for the Apply webpage.


  1. Shadow Effect
    • Effect: The shadow gives a slight 3D effect, lifting the form off the page visually. On hover, the shadow deepens.
    • Benefit: This visual cue indicates interactivity and makes the form stand out, attracting user attention.
  2. Label Formatting
    • Effect: Labels are clearly displayed as block elements with bold text, providing clear associations with their corresponding fields.
    • Benefit: This enhances accessibility and usability, making it easier for users to understand what information is required.
  3. Input Fields
    • Effect: Input fields are spacious, with rounded corners and soft colors, providing a comfortable area for user interaction.
    • Benefit: Enhanced usability encourages users to fill out the form without feeling constrained, reducing frustration.
  4. Button Hover Effects
    • Effect: A color change and slight scaling on hover signals interactivity.
    • Benefit: This feedback encourages clicks and improves overall user engagement.
